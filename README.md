Android TV 平台媒体播放器，用于轻度替代KODI。

1. 全能播放与网络支持：专为电视设计，核心基于 ExoPlayer，流畅播放 4K/HDR 视频。完美支持 SMBv2/v3、NFS、WebDAV 等主流网络协议，轻松直连 NAS 与电脑共享。
2. 极致字幕体验：提供精细的字幕自定义功能（字体导入、大小位置调整、黑边填充模式），让字幕不再遮挡画面。
3. 智能管理与多端同步：内置 TMDB 刮削器，自动匹配电影/剧集海报与元数据。支持局域网内多设备数据同步，无论是客厅还是卧室，播放历史与收藏夹实时互通，无缝断点续播。
4. 支持剧集自动跳过片头时长设定，可对视频文件自动获取影片介绍以及剧集单集内容介绍等影片信息。
